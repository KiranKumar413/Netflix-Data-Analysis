# Netflix Data Analysis 📊

## 📌 Project Overview
This project analyzes Netflix content to identify trends in movies and TV shows, genres, ratings, and country-wise distribution.

## 🛠️ Tools Used
- Python (Pandas, NumPy)
- Matplotlib & Seaborn
- Power BI

## 📊 Key Insights
- Movies dominate Netflix content (~76%)
- Content addition increased rapidly after 2015
- TV-MA is the most common rating
- United States has the highest content
- Documentaries and Stand-Up Comedy are top genres

## 📂 Dataset
Dataset sourced from Kaggle.
